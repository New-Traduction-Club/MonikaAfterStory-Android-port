# Unofficial MAS Port

This project is an unofficial Android port of Monika After Story (MAS).

## Important Notes

> **This repository does not contain game assets, only renpy scripts and the android project.**

- This is a fork or original RAPT project of Ren'Py.

### Unrpa

We made an unrpa tool in kotlin for this port, we are going to use it later...

Thanks to Lattyware for the main logic of unrpa for python, we based the code in: https://github.com/Lattyware/unrpa
